# Cars-Repairs-CPT

This plugin creates a custom post type called 'cars' which allows users to enter video links for repair videos and descriptions of the cars. It also allows users to enter video links for the following car parts: 

* Brand 
* Fuel 
* Video URL for review 
* Year 
* Power steering fluid 
* Tires and wheels 
* Wiper blade front 
* Fluid leaks 
* Fluid check 
* Coolant antifreeze 
* Clock 
* Check engine light 
* Brake fluid 
* Belts 
* Brake light 
* Center brake light 
* Daytime running light 
* Fog light 
* Headlight 
* Highbeam 
* Parking light 
* Reverse light 
* Tail light 
* Turn signal front 
* Turn signal rear 
* Clean battery and terminals 
* Replace battery 
* Jump start 
* Airconditioner 
* Air filter engine 
* Air filter cabin 
* Fix leaks 
* Check oil
